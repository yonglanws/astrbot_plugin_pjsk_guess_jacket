# PJSK 猜曲绘插件

## 功能简介

本插件使用《初音未来 缤纷舞台》的歌曲资源制作猜曲绘游戏，用户根据处理后的封面图片猜出歌曲名称

歌曲资源截至日服5.5周年前

### 特色功能

- 3种不同的图片效果处理（模糊、随机裁切、损坏效果）
- 所有效果均可配置启用/禁用和参数
- 白色色调的精美界面设计
- 支持中文/日文双语显示
- 完整的排行榜系统
- 每日游戏次数限制
- 游戏冷却机制
- 支持模糊匹配和别名识别

## 指令列表

### 猜曲绘游戏

| 指令              | 别名                     | 说明               |
| --------------- | ---------------------- | ---------------- |
| `猜曲绘`           | `pjsk猜曲绘`、`曲绘猜曲`、`猜封面` | 开始一轮猜曲绘游戏        |
| `猜曲绘分数`         | `pjsk猜曲绘分数`、`曲绘猜曲分数`   | 查看自己的游戏数据统计      |
| `猜曲绘排行榜`        | `猜曲绘排行`                | 查看总分排行榜          |
| `猜曲绘自定义名称 [名称]` | `猜曲绘昵称`                | 设置自定义ID（不带参数可清除） |

### 帮助与管理

| 指令       | 说明              |
| -------- | --------------- |
| `猜曲绘帮助`  | 显示帮助信息          |
| `刷新本地数据` | 重新加载本地别名数据（管理员） |

## 配置说明

### 基础配置

| 配置项                     | 类型   | 默认值 | 说明               |
| ----------------------- | ---- | --- | ---------------- |
| `answer_timeout`        | int  | 30  | 答题超时时间（秒）        |
| `daily_play_limit`      | int  | 10  | 每日游戏次数上限（-1为无限制） |
| `game_cooldown_seconds` | int  | 30  | 游戏冷却时间（秒）        |
| `ranking_display_count` | int  | 10  | 排行榜显示人数          |
| `super_users`           | list | \[] | 管理员用户ID列表        |
| `blacklist`             | list | \[] | 黑名单用户ID列表        |
| `group_whitelist`       | list | \[] | 群聊白名单（为空则所有群可用）  |

### 效果配置 (effects)

在配置文件的 `effects` 部分，可以自定义各种图片效果的启用状态和参数，例如：

| 配置项           | 类型   | 默认值  | 说明          |
| ------------- | ---- | ---- | ----------- |
| `enabled`     | bool | true | 是否启用此效果     |
| `difficulty`  | int  | 2    | 猜对时获得的分数    |
| `blur_radius` | int  | 20   | 模糊半径，值越大越模糊 |

难度越高，答对获得的分数越多。

## 答案匹配规则

插件支持多种答案匹配方式：

1. **精确匹配**：输入与歌曲中文名、原名或别名完全一致
2. **模糊匹配**：支持部分字符匹配（相似度≥60%）
3. **容错匹配**：自动处理常见拼写错误（如 l/1/i, o/0 等）
